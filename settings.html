Transcription

Language
Auto-detect identifies the language each time you dictate. Switching away from English may download a multilingual model (~1.6 GB) on Apply. ".en" models are English-only.
Formatting mode
Code mode skips prose cleanup and auto-capitalization; symbol words only convert here, never in Clean or Raw.
Performance
Auto chooses the fastest working local backend on this machine (NVIDIA CUDA when present, else CPU). AMD / Intel GPU (Vulkan) runs on any Vulkan GPU and is much faster than CPU; selecting it downloads a small GPU engine + model once (one-time, offline afterward). If it can't start, ROAR falls back to CPU automatically. Nothing ever leaves your machine.
