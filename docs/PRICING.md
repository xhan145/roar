# Pricing

ROAR is private Windows dictation you own.

Core is free.
Pro is $19 once.
Developer Pack is $29 once.
Supporter is $49 once.

No subscription.
No account.
No cloud transcription.
No telemetry.

## What each edition is for

- **Core (free)** — private, local, offline dictation. Push-to-talk, hands-free,
  streaming preview, multilingual, history, all privacy controls, basic
  vocabulary, scratch-that.
- **Pro ($19 once)** — smarter local dictation: smart formatting, snippet packs,
  vocabulary suggestions, history filters/tags, advanced cleanup, settings
  import/export, advanced milestones.
- **Developer Pack ($29 once)** — a code-aware voice layer: code mode, symbol
  dictation, app profiles, per-app language, project vocabulary, file tagging,
  developer snippet packs.
- **Supporter ($49 once)** — everything in Developer, and supports continued
  development.

Buy once, use forever. Includes all v1.x updates. Optional paid major upgrades
may happen later, but purchased versions keep working.

Core dictation, offline use, privacy controls, and history/audio deletion remain
free.
